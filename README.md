
# 什么值得买 autojs脚本 自动京东关注店铺送京豆

安卓 autojs脚本，根据什么值得买APP上总结的关注京东店铺送京豆页面，自动关注店铺，领取京豆

测试机型 华为 P30

Autojs版本  4.0.1

因为按钮“点击关注”这里采用的是 autojs中区域坐标点击的方式，所以使用其他手机可能需要修改这个位置的具体区域位置
